== 컨테이너의 기초에 대해 알아보겠습니다.

이 튜토리얼에서는 컨테이너를 사용하여 애플리케이션을 쉽게 배포하고 관리하는 방법을 보여줍니다.

[.text-center.strong]
== 컨테이너란 무엇인가?

Linux 컨테이너는 전체 런타임 환경(실행에 필요한 모든 파일)과 함께 애플리케이션을 패키징하고 격리할 수 있는 기술입니다. 
이를 통해 모든 기능을 유지하면서 컨테이너화된 애플리케이션을 다양한 환경 (개발, 테스트, 프로덕션 등) 간에 쉽게 이동할 수 있습니다.
